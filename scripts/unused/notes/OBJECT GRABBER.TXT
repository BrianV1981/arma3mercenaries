1. Build your composition in Eden.
2. Aim the eden camera on the center of your composition.

3. Open debug console and run ObjectsGrabber something like so:

t = [ screenToWorld [0.5,0.5], 50, true] call BIS_fnc_objectsGrabber

or

[getPos player, 50] call BIS_fnc_objectsGrabber

4. Create and open a new .sqf file. Press ctrl + V in it, and it will post all of your object code.
5. To spawn in the composition, use the following code:

0 = [positionWhereYouWantToSpawnYourComposition, azimutOfYourComposition, call (compile (preprocessFileLineNumbers "yourComposition.sqf"))] call BIS_fnc_ObjectsMapper;
